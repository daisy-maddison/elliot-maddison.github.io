Grid Dungeon Generator

This code generates a dungeon by setting an integer array, which it uses as a grid. 1 = a floor, 0 = a wall. It begins by creating two box rooms and then finding a path from room 1 to room 2, filling in floor tiles as it goes. This is repeated for X number of room pairs, which then intersect and create a floor for a procedurally generated dungeon.

Made with: Unity
Language: C#




DirectX UI Class

This code creates a class for handling UI elements in a DirectX program, as well as classes that inherit from it - Text, Image and Button - to make it much easier to implement UI elsewhere. The classes allow you to change the text, sprite of an image, the colour of an image - both the default colour and the highlighted colour - and update their positions as the viewport changes. Additionally, the margin of text inside of a button can be adjusted, so it can move around independent of the button's position.

Made with: DirectX
Language: C++




Scripted Cutscenes

This code allows an object to be manipulated, specifically for use by additional code that handles a cutscene within a game. It includes the ability to transform any object by passing the positions of target objects to a wanted function - such as LookAt, which creates a smooth rotation over time to face the position, or MoveTo which moves the object over time towards the target position.

Made with: Unity
Language: C#




DVD Logo

This code creates a small window that contains a single sprite of the DVD logo, which moves in a straight line across the screen. When it hits the egde, it bounces and changes to a random colour. This goes on indefinitely or until the user exits.
